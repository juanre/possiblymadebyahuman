# web app

Future Vite React record-viewing app.

## Responsibility

- Public `/<short_signature>` record page.
- Standing disclaimer, capture-context summary, quick stats, content-blind process timeline, analyzer signal cards, and verification panel.
- Browser-side chain verification using `packages/format` when implemented.

## Non-responsibility

- Marketing/docs/blog pages.
- Ingestion and storage.
- Producer capture.
- Rendering plaintext in the public deployment.
- Human/AI verdicts, confidence percentages, or certification-style badges.

M0 is scaffold only. Record UI implementation begins in M4.
