# site

Future Hugo static site for landing, docs, and blog pages.

## Responsibility

- Public `/`, `/docs/*`, and `/blog/*` routes.
- Product explanation, installation/use docs, privacy model, verification docs, producer conformance docs, and threat model.
- Candid lightweight tone: “we cannot prove it, but here is us caring enough to show the work.”

## Non-responsibility

- Public record pages such as `/<short_signature>`.
- Backend ingestion APIs.
- Producer capture.
- Detector/certificate language or humanness scoring.

M0 is scaffold only. Static site implementation begins in M5.
